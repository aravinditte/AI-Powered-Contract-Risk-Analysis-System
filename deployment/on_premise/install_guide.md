# On-Premise Installation Guide

## Supported Environment
- OS: Ubuntu 22.04 LTS
- CPU: x86_64
- Network: Internal LAN only

## Installation Steps

1. Install Docker & Docker Compose
2. Clone the repository to internal server
3. Configure environment variables
4. Run docker-compose up -d
5. Validate system via /docs endpoint

## Notes
- External internet access is NOT required
- AI models are loaded from local storage
- No data is sent outside the organization
